Stroop Task

The Stroop Task depends on our human tendency to read words very quickly, almost automatically. It seems that we do this more easily than we can name colors that we see. The Stroop Task (named for the psychologist who discovered this effect in the 1930's) exploits this tendency by showing words which are the names of colors (Red, Green, Blue, Yellow), but printing them in colors different than the color named, for example Blue. Because of our tendency to read the words automatically, you must concentrate hard in order to quickly name the actual color which is shown instead of responding with the word which was printed. Try to respond as quickly (but correctly) as you can.

스트룹 검사 (Stroop Task)

스트룹 검사는 사람들이 단어를 매우 빨리, 거의 의식하지 않고 읽어내는 경향에 의한 검사입니다. 대개 사람들은 보여지는 색깔의 이름을 붙이는 것보다 단어를 읽는 것을 더 쉽게 해내곤 합니다. '스트룹 검사'는, 1930년대에 이 효과를 발견한 심리학자의 이름을 따 만들어진 것입니다. 스트룹은 색깔의 이름(빨강, 초록, 파랑, 노랑)과 함께 색깔을 표시하되, 이름과 다른 색을 표시하여 테스트를 함으로써 이러한 경향을 밝혀내었습니다(예를 들어, 파랑). 우리가 단어를 무의식중에 읽어내는 경향으로 인해, 여러분들은 표시된 이름 대신 실제 색깔 의 이름을 말하는 것에 좀 더 집중하셔야 할 것입니다. 가능한한 정확하고 빠르게 응답해보세요.
